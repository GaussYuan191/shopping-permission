#shopping-permission

## 电子商城权限管理后台-后端部分

## 简介

本项目基于Springboot+MybatisPlus的快速开发脚手架，拥有完整的权限管理功能。

## 技术栈

| 技术                   | 版本    | 说明             |
| ---------------------- | ------- | :--------------- |
| SpringBoot             | 2.3.0   | 容器+MVC框架     |
| SpringSecurity         | 5.3.2   | 认证和授权框架   |
| MyBatis                | 3.5.4   | ORM框架          |
| MyBatis-Plus           | 3.3.2   | MyBatis增强工具  |
| MyBatis-Plus Generator | 3.3.2   | 数据层代码生成器 |
| Druid                  | 1.1.10  | 数据库连接池     |
| JWT                    | 0.9.0   | JWT登录支持      |
| Lombok                 | 1.18.12 | 简化对象封装工具 |
